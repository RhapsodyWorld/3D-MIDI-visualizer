# 3D MIDI visualizer
3D MIDI visualization for web using Three.js and WebGL.<br>
Built on the <a href="https://github.com/gasman/jasmid" target="_blank">MIDI file reader library</a> by <a href="https://github.com/gasman">gasman</a>.
